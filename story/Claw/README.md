# Claw - 中文网页构建器

基于 GrapesJS 的可视化网页构建工具，全中文界面，拖拽式操作。

## 功能特点

- ✅ **全中文界面** - 所有菜单、按钮、组件都已汉化
- ✅ **拖拽式操作** - 简单直观的拖放界面
- ✅ **丰富的组件库** - 包含文本、图片、表单、布局等组件
- ✅ **实时预览** - 所见即所得的编辑体验
- ✅ **响应式设计** - 支持桌面、平板、手机预览
- ✅ **导出功能** - 一键导出完整的 HTML/CSS 代码
- ✅ **本地存储** - 自动保存项目，随时继续编辑
- ✅ **完全免费** - 开源软件，无需付费

## 安装使用

### 1. 安装依赖

```bash
npm install
```

### 2. 启动服务

使用任何本地服务器，例如：

```bash
# 使用 Python 3
python -m http.server 8000

# 使用 Node.js http-server
npx http-server

# 使用 PHP
php -S localhost:8000
```

### 3. 打开浏览器

访问 `http://localhost:8000`

## 组件分类

### 基础组件
- 文本、标题、段落
- 链接、分隔线
- 引用、代码块
- 列表（有序、无序）

### 媒体组件
- 图片、视频

### 表单组件
- 各种样式按钮
- 文本输入框、邮箱、密码
- 文本域、复选框、单选框
- 下拉选择、完整表单

### 布局组件
- 容器、区块
- 两列布局、三列布局

### 页面组件
- 页头、页脚
- 主视觉区、卡片

## 主要功能

### 1. 组件拖放
- 从左侧组件库拖拽组件到画布
- 支持拖放排序和嵌套

### 2. 样式编辑
- 点击画布上的元素
- 在右侧样式面板调整样式
- 支持 CSS 属性完整编辑

### 3. 属性编辑
- 在右侧属性面板设置元素属性
- 支持 ID、类名、标题等属性

### 4. 设备预览
- 顶部工具栏切换设备
- 桌面、平板、手机三种预览模式

### 5. 导出代码
- 点击"导出模板"按钮
- 自动下载完整的 HTML 文件
- 包含完整的 CSS 样式

## 快捷键

- `Ctrl+Z` / `Cmd+Z` - 撤销
- `Ctrl+Y` / `Cmd+Y` - 重做
- `Ctrl+S` / `Cmd+S` - 保存
- `Ctrl+E` / `Cmd+E` - 导出

## 自定义开发

### 添加新组件

在 `js/blocks.js` 中添加新的组件定义：

```javascript
'my-component': {
    label: '我的组件',
    content: '<div>组件内容</div>',
    category: '基础组件'
}
```

### 修改样式

编辑 `css/custom.css` 文件自定义界面样式。

### 扩展功能

在 `js/app.js` 中添加新的功能模块。

## 常见问题

### Q: 导出的 HTML 可以直接使用吗？
A: 是的，导出的 HTML 是完整的，可以直接部署到任何服务器。

### Q: 支持哪些浏览器？
A: 支持所有现代浏览器，推荐使用 Chrome。

### Q: 如何保存项目？
A: 项目会自动保存到浏览器的本地存储中。

### Q: 可以导入现有的 HTML 吗？
A: 可以通过修改 `js/app.js` 中的初始 HTML 内容来导入。
